# LED sequencer effect

This is a simple LED circuit using Transitor-Transistor-Logic circuit and through this it will make an LED go up one by one. I'm building an astable circuit because of the fact that I want the LED to light up for a specific duration. I want it to act like the lights are bouncing like on FRC LED lights!

## March 27, 2026 - Researching astable circuits

Today I worked on using the template as a draft and experimenting with it. Initially I wanted it to be monostable but I don't think a fixed duration would achieve that effect when I read through the Resolution Hardware Document. Through my research I learned about astable circuits and they can change brightnesses on there own as long as it is powered. I found a schematic from ElecCircuit.com of astable circuit led sequence.
